
### Airline Revenue Management Analytics Dashboard

## Project Overview
This project demonstrates an end‑to‑end data analytics workflow applied to airline revenue management. 
The objective is to analyze airline transaction data to identify passenger demand patterns, route performance, ticket pricing trends, and revenue optimization opportunities.

The project integrates SQL, Python, and Power BI to perform data analysis, forecasting, and visualization.

---

## Objectives
- Analyze passenger demand and seasonal travel patterns.
- Evaluate airline route performance and revenue contribution.
- Examine ticket pricing trends and fare class revenue distribution.
- Calculate key airline revenue management KPIs.
- Forecast future passenger demand using time‑series analysis.
- Build an interactive dashboard for business insights.

---

## Dataset
The dataset contains **120,000+ airline transaction records** including:

- Date
- Airline
- Origin
- Destination
- Route
- Fare Class
- Booking Channel
- Capacity
- Passengers
- Ticket Price
- Revenue

---

## Tools & Technologies
- SQL (MySQL)                   – Data extraction and aggregation
- Python (Pandas, NumPy, ARIMA) – Data analysis and demand forecasting
- Power BI                      – Interactive dashboard and visualization
- Excel                         – Data exploration

---

## Key Metrics (KPIs)
The following airline revenue management KPIs were calculated:

- Total Revenue
- Total Passengers
- Average Ticket Price
- Load Factor (Passengers / Capacity)
- Yield (Revenue per Passenger)

These metrics help evaluate airline operational performance and revenue efficiency.

---

## Data Analysis

## SQL Analysis
SQL was used to perform:
- Revenue aggregation by route
- Passenger demand analysis
- Average ticket price by route
- Load factor calculation
- Monthly revenue trend analysis

## Python Analysis
Python was used for:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Seasonal demand analysis
- Time‑series forecasting using ARIMA

Passenger demand forecasting was performed to predict future passenger volumes.

## Dashboard Features (Power BI)

The interactive dashboard includes:

### Revenue Overview
- Total Revenue
- Total Passengers
- Average Ticket Price
- Load Factor

### Demand Trends
- Passenger Demand by Month
- Revenue Trend by Month

### Route Performance
- Passenger Demand by Route
- Average Ticket Price by Route

### Fare Class Analysis
- Revenue distribution by Fare Class

### Filters (Slicers)
- Airline
- Route
- Fare Class

These filters allow users to analyze revenue performance across different markets.

---

## Key Insights
- Passenger demand shows seasonal fluctuations, with peak travel periods generating the highest     revenue.
- Certain routes demonstrate significantly higher passenger demand and revenue contribution.
- Business class generates higher revenue per passenger, highlighting its strong yield potential.
- Demand forecasting indicates stable passenger volumes, supporting strategic pricing and capacity planning.
